a. ¿Quién inventó el sistema de control de versión Git y por qué?

Por lo que recuerdo, Git surge debido a que el proyecto Linux Kernel usaba el sistema de versionado BitKeeper (en su entonces, gratis), 
y cuando sacaron el permiso gratis de la aplicación, el equipo de Linux Kernel se puso en desarrollo de un sistema de versionado que tuviera servicio gratuito.
