a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló 
originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005.
b. ¿A quién pertenece actualmente Github y por qué?
Se utiliza principalmente para la creación de código fuente de programas de ordenador. 
El software que opera GitHub fue escrito en Ruby on Rails. Desde enero de 2010, GitHub opera bajo el nombre de GitHub, Inc.
c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
Si, con Github Desktop.
